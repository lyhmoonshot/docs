Kimi 开放平台提供三个模型系列。完整的模型列表和描述请参阅[模型列表](/docs/models)。

## 参数对比

不同模型系列对 Chat Completions API 参数有不同的默认值和约束：

| 参数 | kimi-k2.5 | kimi-k2 系列 | kimi-k2-thinking 系列 | moonshot-v1 系列 |
|------|-----------|-------------|----------------------|-----------------|
| `temperature` | **不可修改** | 0.6 | 1.0 | 0.0 |
| `top_p` | 0.95 **不可改** | 1.0 | 1.0 | 1.0 |
| `n` | 1 **不可改** | 1（最大 5） | 1（最大 5） | 1（最大 5） |
| `presence_penalty` | 0 **不可改** | 0（可修改） | 0（可修改） | 0（可修改） |
| `frequency_penalty` | 0 **不可改** | 0（可修改） | 0（可修改） | 0（可修改） |
| `thinking` | 支持 | — | — | — |

<Note>
当 `temperature` 接近 0 时，`n` 只能为 1，否则将返回 `invalid_request_error`。
</Note>
