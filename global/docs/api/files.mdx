## Upload file

<Note>
  Each user can upload a maximum of 1,000 files. Each file must not exceed 100 MB, and the total size of all uploaded files must not exceed 10 GB. The file parsing service is currently free, but rate limiting may be applied during peak traffic periods.
</Note>

### Request endpoint

```text
POST https://api.moonshot.ai/v1/files
```

Once the file is uploaded successfully, the platform will process it according to the selected `purpose`.

### Example request

#### Python example

```python
# The file can be of various types
# The purpose currently supports "file-extract", "image", and "video"
file_object = client.files.create(file=Path("xlnet.pdf"), purpose="file-extract")
```

For `purpose="file-extract"`, the file contents will be extracted. You can also use `purpose="image"` or `purpose="video"` to upload images and videos for vision tasks.

### Supported formats

The file interface supports the same file formats as file upload in the Kimi assistant. Supported formats include `.pdf`, `.txt`, `.csv`, `.doc`, `.docx`, `.xls`, `.xlsx`, `.ppt`, `.pptx`, `.md`, `.jpeg`, `.png`, `.bmp`, `.gif`, `.svg`, `.svgz`, `.webp`, `.ico`, `.xbm`, `.dib`, `.pjp`, `.tif`, `.pjpeg`, `.avif`, `.dot`, `.apng`, `.epub`, `.tiff`, `.jfif`, `.html`, `.json`, `.mobi`, `.log`, `.go`, `.h`, `.c`, `.cpp`, `.cxx`, `.cc`, `.cs`, `.java`, `.js`, `.css`, `.jsp`, `.php`, `.py`, `.py3`, `.asp`, `.yaml`, `.yml`, `.ini`, `.conf`, `.ts`, `.tsx`, and more.

<Tip>
  When uploading a file, use `purpose="file-extract"` if you want the model to use the extracted file contents as context.
</Tip>

### Multi-file chat example

If you want to upload multiple files and have a conversation with Kimi based on these files, you can use the following pattern:

```python
from typing import *

import os
import json
from pathlib import Path

from openai import OpenAI

client = OpenAI(
    base_url="https://api.moonshot.ai/v1",
    api_key=os.environ["MOONSHOT_DEMO_API_KEY"],
)


def upload_files(files: List[str]) -> List[Dict[str, Any]]:
    """
    Upload each file, extract its content, and convert it into a system message.
    """
    messages = []

    for file in files:
        file_object = client.files.create(file=Path(file), purpose="file-extract")
        file_content = client.files.content(file_id=file_object.id).text
        messages.append({
            "role": "system",
            "content": file_content,
        })

    return messages


def main():
    file_messages = upload_files(files=["upload_files.py"])

    messages = [
        *file_messages,
        {
            "role": "system",
            "content": "You are Kimi, an AI assistant provided by Moonshot AI. You are more proficient in Chinese and English conversations. You provide users with safe, helpful, and accurate answers. You will refuse to answer any questions related to terrorism, racism, pornography, or violence. Moonshot AI is a proper noun and should not be translated into other languages.",
        },
        {
            "role": "user",
            "content": "Summarize the content of these files.",
        },
    ]

    print(json.dumps(messages, indent=2, ensure_ascii=False))

    completion = client.chat.completions.create(
        model="kimi-k2-turbo-preview",
        messages=messages,
    )

    print(completion.choices[0].message.content)


if __name__ == '__main__':
    main()
```

## List files

<Note>
  Use this endpoint to list all files uploaded by the current user.
</Note>

### Request endpoint

```text
GET https://api.moonshot.ai/v1/files
```

### Example request

```python
file_list = client.files.list()

for file in file_list.data:
    print(file)  # Inspect the metadata of each file
```

## Delete file

<Note>
  Use this endpoint to delete files that are no longer needed.
</Note>

### Request endpoint

```text
DELETE https://api.moonshot.ai/v1/files/{file_id}
```

### Example request

```python
client.files.delete(file_id=file_id)
```

## Get file content

<Note>
  This endpoint retrieves extraction results for files uploaded with `purpose="file-extract"`.
</Note>

- The returned content is typically a valid JSON-formatted string that follows the platform's recommended extraction format.
- If you need to extract multiple files, you can concatenate them into a single large string separated by newline characters `\n`, then add that content to conversation history as `system` messages.

### Request endpoint

```text
GET https://api.moonshot.ai/v1/files/{file_id}/content
```
